# MathOnco_Intro
Code for introductory course on mathematical oncology centered on linking models with experimental data. 
CAT Data comes from Craig et al. PLOS Comp Biol, 2019 10.1371/journal.pcbi.1007278
Kavran data comes from Kavran et al., PNAS, 2022, 10.1073/pnas.2113535119
JITC Data is the data used in Jenner et al., JITC, 2021, 10.1136/jitc-2020-001387
Friberg data is simulated neutrophil counts following chemotherapy and was used in Cassidy et al. Bull Math Biol,
